# 🛡️ Web3 Risk Monitor - AI Analysis & History

A robust Full-Stack solution developed by **Agência IA Diniz** to monitor and analyze Smart Contract risks in real-time. This project combines AI logic with Blockchain data and persistent storage.

## 🚀 The Project
This MVP (Minimum Viable Product) was built to help users identify potential threats in Ethereum, Polygon, and Sepolia contracts. It features a modern UI, real-time wallet connection, and a centralized database for analysis history.

## 🛠️ Tech Stack
- **Frontend:** React + Vite, Tailwind CSS, Lucide Icons.
- **Web3 Interface:** Wagmi, Viem, RainbowKit.
- **Backend:** Node.js, Express, TypeScript.
- **Database:** Supabase (PostgreSQL) with RLS (Row Level Security).
- **Global Reach:** i18next (Dual language support: EN/PT).

## 📂 Architecture
- `/frontend`: User interface and Web3 integrations.
- `/backend`: API to manage history and secure database communication.
- `/core`: Shared business logic and AI analysis rules.
